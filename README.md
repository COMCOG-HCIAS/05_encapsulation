
# Anaphoric Encapsulation: Reading time & Discourse particle analysis

## Files

- `1_descriptionReport`: original data description (2024)

- `1_descriptionReport_new`: data description with updated data (2025/2026)

- `2_mainReport`: original analysis with data from 2024

- `2_mainReport_AOI16`: original analysis with *original data*, reduced to only AOI 1 and AOI 6 and *including* group-wise comparisons

## Dependencies

The reports use the helper package `DPKogHelpers` that is currently not publicly available. If you want to reproduce the analyses, please contact us and we will provide the package source code.
